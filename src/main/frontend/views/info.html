About Groovy Playground

This is a simple editor, where you can play around with Groovy. Just enter some Groovy script in the editor and press Execute (or Control + Enter).

On the right panel you will see the output of the script and its return value. You can hover over the output to highlight the lines in the editor, that generated that output. This only works as long as you don't edit the output after execution.

Each time you execute your code it will be saved automatically and will be restored when you return to this page.

Known issues

  • System.out and System.err are not working (only print/ln)
  • println is only working in the root of the script (and not if you use it inside a class)

You can find the list of all open issues on the GitHub Page.
